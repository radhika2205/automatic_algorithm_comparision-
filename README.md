# Automatic Algorithm Comparison

## 📌 Overview

This project compares multiple Machine Learning algorithms on a dataset and evaluates their performance to identify the best model.

## 🎯 Objective

* Train multiple ML models
* Compare their performance
* Select the best-performing algorithm

## ⚙️ Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn

## 📂 Project Structure

```
automatic_algorithm_comparision/
│── app.py
│── automatic_algorithm_comparision.py
│── requirements.txt
│── README.md
```

## 🚀 Features

* Automatic model training
* Performance comparison
* Simple and easy to run

## ▶️ How to Run

1. Clone repository:

```
git clone https://github.com/radhika2205/automatic_algorithm_comparision.git
```

2. Install dependencies:

```
pip install -r requirements.txt
```

3. Run project:

```
python app.py
```

## 📊 Output

* Displays comparison of different algorithms
* Helps select best model based on performance

## 📌 Note

You can modify the dataset or add more algorithms as needed.
